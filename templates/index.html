Project Overview

What the platform delivers

The application predicts property prices from meaningful housing attributes including bedrooms, bathrooms, square footage, floors, waterfront availability, view score, condition, and year details. Instead of stopping at the machine learning model, the project extends into a complete product experience that a recruiter, mentor, or client can explore directly.

Build Journey

How I built the project step by step

The development approach followed a practical ML product pipeline, moving from data preparation to model deployment and a polished front-end presentation.

01

Problem Framing

I defined the goal clearly: predict house prices from structured property features in a reliable and understandable way.

02

Data Preparation

I explored the dataset, handled feature selection, and identified the strongest variables for meaningful price estimation.

03

Model Training

I trained a regression model to learn the relationship between property characteristics and final market value.

04

Model Export

The trained model was serialized with Pickle so it could be loaded quickly inside the Flask application.

05

Web Designing

I designed a responsive interface with stronger visual hierarchy, premium layouts, and content sections that explain the project professionally.

06

Authentication and Delivery

I connected Supabase-based login and signup flows so the project feels like a complete web product instead of a single form page.
